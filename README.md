# 🛒 E-Commerce Marketplace Analytics & Sales Prediction

## 📌 Overview

This project focuses on analyzing e-commerce sales data and building a machine learning model to predict sales, combined with an interactive dashboard for business insights.

It simulates a real-world marketplace environment helping identify trends in category performance, seller efficiency, and revenue distribution.

---

## 🎯 Objectives

* Analyze sales data to uncover patterns and trends
* Predict sales using machine learning models
* Build an interactive dashboard for decision-making
* Replicate real-world marketplace analytics

---

## ⚙️ Tech Stack

* **Python** (Pandas, NumPy, Scikit-learn)
* **SQL** (data extraction & analysis)
* **Excel** (data cleaning & preprocessing)
* **Power BI** (dashboard & visualization)

---

## 🤖 Machine Learning Models

* Linear Regression
* Random Forest
* XGBoost

### 📊 Evaluation Metrics

* R² Score
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)

---

## 📊 Dashboard (Power BI)

An interactive dashboard was built to visualize marketplace performance and key business metrics.

### Key Features:

* KPI cards for **Total Sales, Profit, Orders**
* Category-wise and region-wise performance analysis
* Time-based sales trends
* Top-performing products and segments
* Interactive filters (category, region, date)

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Chrish143/sales-prediction-ml.git
cd sales-prediction-ml
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the Python script:

```bash
python src/main.py
```
